# Sheets_Sleep-Screen-time-and-Stress

Dataset found on Kaggle (CC0: Public Domain):
https://www.kaggle.com/datasets/jayjoshi37/sleep-screen-time-and-stress-analysis/data?select=sleep_mobile_stress_dataset_15000.csv

This file contains 15000 records analyzing the relationship between smartphone usage, sleep patterns & stress levels. The dataset includes demographic information as well as screen time behavior, sleep duration, sleep quality indicators, & lifestyle factors like physical activity & caffeine intake.

The analysis has been done entirely on google sheets & the results can be found in the pdf file.
